### 题目要求
将一个按照升序排列的有序数组，转换为一棵高度平衡二叉搜索树。

本题中，一个高度`平衡`二叉树是指一个二叉树每个节点 的`左右`两个子树的高度差的绝对值`不超过` 1。

### 题目示例
- **`示例:`**
```
给定有序数组: [-10,-3,0,5,9],

一个可能的答案是：[0,-3,9,-10,null,5]，它可以表示下面这个高度平衡二叉搜索树：

      0
     / \
   -3   9
   /   /
 -10  5
```
